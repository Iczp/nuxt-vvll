## 流程图 flowchart

```md
flowchart LR
Start --> Stop
```

```mermaid
flowchart LR
Start --> Stop
```

```mermaid
flowchart TD
Start ==> Stop
```

### 方向

可能的流程图方向是：

- `TB` - 从上到下
- `TD` - 自上而下/与自上而下相同
- `BT` - 从下到上
- `RL` - 右到左
- `LR` - 左到右
